Code

Dr Racket is different from other languages because it has a definitions window and a interactions window. In the definitions window, you would basically put methods and functions there and in the interactions window you would call the methods you defined in the definitons window.

Implimentation

Dr Racket will show the image in the iteractions window instead of creating a whole new window. If you wanted, you can continue calling as many functions as you want.
If you really wanted, you can write the funtion in the interactions area. The definitions area are usually for more complicated funtions.
If you wrote it in the interations area, it would be: ( define the-circle (circle 200 "solid" "blue"))
